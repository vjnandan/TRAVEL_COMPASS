# Travel Compass — Tourist Guide System

A web application that helps tourists discover destinations and build 
personalized travel itineraries — built as a BCA project at Christ University.

## Features
- Browse and search tourist destinations by location, category, or budget
- Build a day-by-day itinerary across multiple destinations
- Submit and view reviews/ratings for places
- Admin panel for managing destination content and moderating reviews

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **External API:** Google Maps API (location display)
